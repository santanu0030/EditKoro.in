Mini Project: Portfolio Page for "Editkaro.in"

Objective: 
Create an interactive and dynamic portfolio webpage for a real-world client, 
Editkaro.in—a social media marketing and video editing agency. You will showcase their 
work, which includes categories like short-form videos, long-form videos, gaming videos, 
football edits, eCommerce ads, documentary style, color grading, anime videos, and ads.

Requirements: 
• Design a visually engaging portfolio webpage. 
• Use HTML, CSS, and JavaScript to implement interactivity and responsiveness. 
• Include creative elements like video previews, filters, or animations. 
• Organize videos by categories with clear navigation. You can add a categories or 
filter by button or simple display all of them with headings, you your creativity. 
• Ensure the webpage is fully responsive for mobile and desktop views.

Instructions: 
• You can use your own ideas and creativity to design the portfolio page. Feel free to 
add elements like hover effects, video lightboxes, or unique layouts. 
• Best Project Incentive: The best-designed portfolio page will be shared with the 
client and featured on VaultofCodes' website. The creator will also be tagged on 
social media platforms, giving you public recognition for your work.
